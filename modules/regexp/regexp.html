Регулярные выражения

Метод String.match

String.match(regexp)

Строка на вход: "Маленькие зайки в майках, скачут по лужайке"

  • 1. Регулярное выражение /ай/

    Возвращается первое ай "Маленькие зайки в майках, скачут по лужАйке", т.к. нет ключа на повторное использование

  • 2. Регулярное выражение /ай/g

    Возвращается первое и второе ай "Маленькие зайки в майках, скачут по лужАйке", из-за ключа /g, но Ай игнорируется, т.к. не соответствует ключу по регистру

  • 3. Регулярное выражение /ай/gi

    Возвращается первое и второе ай "Маленькие зайки в майках, скачут по лужАйке", учтены все ай и Ай

  • 4. Регулярное выражение /ой/gi

    Возвращает null, т.к. совпадений нет

  • 5. Регулярное выражение /ай/gi, через конструктор

    Результат аналогичен примеру 3
